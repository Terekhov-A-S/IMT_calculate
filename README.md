# Калькулятор расчёта индекса массы тела (ИМТ)

Консольное приложение для расчёта индекса массы тела (ИМТ / BMI) по заданным параметрам роста и веса. Программа помогает оценить соответствие массы человека его росту и определить категорию веса.

## Функциональность

Программа позволяет:
* ввести рост (в сантиметрах) и вес (в килограммах);
* рассчитать индекс массы тела по стандартной формуле;
* получить интерпретацию результата согласно классификации ВОЗ;
* увидеть рекомендацию по корректировке веса (если необходимо).

## Формула расчёта

Индекс массы тела рассчитывается по формуле:

$$
\text{ИМТ} = \frac{\text{вес (кг)}}{(\text{рост (м)})^{2}}
$$

## Классификация ИМТ (по данным ВОЗ)

| Диапазон ИМТ | Категория |
|------------|----------|
| $< 18{,}5$ | Недостаточная масса тела |
| $18{,}5–24{,}9$ | Нормальная масса тела |
| $25{,}0–29{,}9$ | Избыточная масса тела (предожирение) |
| $30{,}0$ и выше | Ожирение |
